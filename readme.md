# Three.js Starter

This repo consists of professional development environment, made to ease static site generation.

## Contents

- Pug template engine
- Sass, postcss, autoprefixer
- JavaScript (ECMAScript 2015+)
- 3D models, shaders, images
- Merging WebGL with DOM
